###动态内存分配
new和delete操作符
```cpp
#include <iostream>

using namespace std;

void main()
{
	double* ptr(nullptr);     //所有指针都应该初始化
	ptr = new double(99.0);   //new操作符应该返回空闲存储器中分配给double变量的内存地址，并在指针ptr中存储该地址.
	cout << *ptr << endl;
	delete ptr;               //当不再需要动态分配的某个变量时，可以用delete操作符将其占用的内存释放到空闲存储器.
}
```
