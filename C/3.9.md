#for语句
```text
for(表达式1;表达式2;表达式3)
	语句;
```
先计算表达式1，通常是为循环变量指定值，在整个循环中，它只执行一次。表达式2，为条件，如果不为0(成立)，继续循环，如果为0(假),终止循环。在表达式2为真时，就去计算表达式3。
```c
#include <stdio.h>

void main()
{	
	int x=1;
	for(x;x<=50;x++)       //或者使用for(;x<=50;x++)
	{
		printf("%d ", x);
	}
	printf("\n");
}
```
#####break语句
break是跳出当前循环体。
```c
#include <stdio.h>

void main()
{	
	int x=1;
	for(x;x<=50;x++)
	{
		printf("%d ", x);
		if(x==10)
		{
			break;
		}
	}
	printf("\n");
}
```
#####continue语句
continue是跳出这一次的循环。
```c
#include <stdio.h>

void main()
{	
	int x=1;
	for(x;x<=50;x++)
	{
		if((x%2) != 0)
		{
			continue;
		}
		printf("%d ", x);
	}
	printf("\n");
}
```
