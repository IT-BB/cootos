#安装Node.js
```bash
wget http://nodejs.org/dist/v0.10.24/node-v0.10.24-linux-x64.tar.gz
tar -zxvf node-v0.10.24-linux-x64.tar.gz
mv node-v0.10.24-linux-x64 /usr/local/node
```
#####设置环境变量
```bash
echo 'export PATH=$PATH:/usr/local/node/bin' >> /etc/profile
```
执行下面的命令，使环境变量生效
```bash
source /etc/profile
```
