###正则表达式

```js
var x = /^[a-zA-Z0-9_\.-]+@[a-zA-Z0-9_-]+(\.[a-zA-Z0-9_-]+)+$/;
```
<table>
	<tr><td>^</td><td>行首匹配</td></tr>
	<tr><td>$</td><td>行尾匹配</td></tr>
	<tr><td>[a-zA-Z0-9_\.-]</td><td>匹配括号中的字符集中的任意字符</td></tr>
	<tr><td>x+</td><td>匹配至少一个x</td></tr>
	<tr><td>()+</td><td>匹配至少一个()</td></tr>
</table>
